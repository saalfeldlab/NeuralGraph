Exploration

Visual Record of Parallel LLM-Guided Connectivity Recovery

This page presents the visual record of the closed-loop exploration: learned connectivity matrices, neural activity traces, UCB exploration trees, connectivity scatter plots, kinographs, and learned signaling functions for each block. Each block introduces a new simulation regime; within each block, the LLM proposes training parameter mutations via UCB tree search with 4 parallel slots per batch.

Current status

Block 23 completed (fill=80%, 30k frames). 276 iterations completed across 23 regimes. n_frames=30k rescues dense and low-gain networks (n=300, n=600, g=3/n=200) but NOT sparse connectivity (subcritical rho) or partial connectivity (fill=80%, conn locked at filling_factor). Two structural limits confirmed. n=1000 mapped at 30k (conn=0.745, needs ~100k). 65 principles established.
